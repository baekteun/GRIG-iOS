# Features

- GSM 학생들의 Github 랭킹을 볼 수 있습니다.
- 리스트의 정렬은 Github 활동, 기수로 옵션을 설정할 수 있습니다.
- 리스트에서 유저를 탭하면 유저의 상세한 정보를 볼 수 있습니다.
- 왼쪽 상당의 불모양 아이콘 또는 유저상세정보에서 Compete버튼을 누르면 유저간 분석표를 보여줍니다.
